Network file sharing - AFS

By Kurt Seifried [email protected]

 

A high end, commercial file sharing protocol suitable for large installations with high security and performance requirements. The FAQ is available at: http://www.angelfire.com/hi/plutonic/afs-faq.html. A free AFS client implementation for a variety of unices (including Linux of course) is available from: http://www.stacken.kth.se/projekt/arla/.
