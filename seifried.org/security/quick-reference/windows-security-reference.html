Microsoft Windows Security Quick Reference Card

By Kurt Seifried, [email protected], http://www.seifried.org/security/quick-reference/windows-security-reference.html

This is a quick reference card for Microsoft Windows security. It consists of several sections, each taking care of a specific problem, typically in under 5 minutes. This guide is designed to help people secure their systems, be they home users, a business with one or more PC's. I encourage you to print this guide out and give it to people experiencing security problems, spyware problems, or who are not completely sure of how to secure a system.

Installing Free Anti-spyware software

Spyware is a critical problem, with some estimates of infected systems running as high as %80. It is virtually guarenteed that any Windows system using email and the web will have Spyware installed, unbeknownst to the user. There are several free Anti-Spyware products, the best two being “Spybot Search and Destroy” and “Ad-Aware”. Additionally Microsoft currently has a free beta product that appears to work quite well, however it is unlikely it will remain free for more then a few months.
