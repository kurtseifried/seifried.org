KSBPA-001 - SSL wrapped IMAP and POP on Linux systems

Many Linux vendors ship software that allows users to SSL wrap IMAP and POP, but very few (none so far) make it easy for administrators to do so.

KSBPA-002 - Cron

Many Linux vendors ship cron and by default allow all users touse cron. Most users do not need access to cron and should be restricted from using it.
